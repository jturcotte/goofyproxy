goofyproxy by jturcotte

goofyproxy acts as an HTTP proxy that filters received data through an external application.

To use it:

  • run it by specifying which external program to use as filter
  • go in network preferences of your browser to configure localhost:PORT as proxy for HTTP connections
  • load your favorite web page
Samples:

ImageMagick's sketch simulation

goofyproxy 8080 image/ convert - -colorspace Gray -sketch 0x20+120 -


ImageMagick's oil paint simulation

goofyproxy 8080 image/ convert - -paint 3 -



Perl to replace text in HTML pages

goofyproxy 8080 text/html perl -nle ' $_=~ s/(?!<.*?)([^\w])cat(?![^<>]*?>)/\1Batman/ig; print $_'
